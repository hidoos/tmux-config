# Tmux 配置快捷键文档

> 前缀键: `Ctrl+s` (替代默认的 `Ctrl+b`)

## 通用操作

| 快捷键 | 说明 |
|--------|------|
| `prefix` + `r` | 重新加载配置文件 |

## 会话管理

| 快捷键 | 说明 |
|--------|------|
| `prefix` + `C-c` | 新建会话 |
| `Alt+Shift+s` | 新建会话 (脚本) |
| `prefix` + `.` | 重命名会话 |
| `prefix` + `W` | 选择树 (会话/窗口浏览) |

## 窗口管理

| 快捷键 | 说明 |
|--------|------|
| `Alt+o` | 新建窗口 (保持当前路径) |
| `Alt+Q` | 关闭当前 pane |
| `Alt+O` | 将当前 pane 拆分为独立窗口 |
| `prefix` + `,` | 重命名窗口 |
| `Alt+s` | 切换 scratchpad |

## 窗口导航

| 快捷键 | 说明 |
|--------|------|
| `prefix` + `C-p` | 上一个窗口 |
| `prefix` + `C-n` | 下一个窗口 |
| `Alt+1` ~ `Alt+9` | 切换到第 1-9 个窗口 |
| `Alt+<` | 与左边窗口交换位置 |
| `Alt+>` | 与右边窗口交换位置 |

## 会话导航

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+1` ~ `Ctrl+9` | 按索引切换会话 |
| `F1` ~ `F5` | 按索引切换会话 (备选) |
| `prefix` + `l` | 向左移动会话 |
| `prefix` + `y` | 向右移动会话 |

## 分割窗口

| 快捷键 | 说明 |
|--------|------|
| `prefix` + `u` | 向上分割 |
| `prefix` + `e` | 向下分割 |
| `prefix` + `n` | 向左分割 |
| `prefix` + `i` | 向右分割 |

## Pane 导航

| 快捷键 | 说明 |
|--------|------|
| `Alt+h` | 选择左边 pane |
| `Alt+j` | 选择下边 pane |
| `Alt+k` | 选择上边 pane |
| `Alt+l` | 选择右边 pane |
| `Alt+f` | 切换 pane 全屏缩放 |
| `prefix` + `>` | 与下一个 pane 交换 |
| `prefix` + `<` | 与上一个 pane 交换 |
| `prefix` + `\|` | 交换 pane |
| `prefix` + `Space` | 切换 pane 布局方向 |

## Pane 调整大小

| 快捷键 | 说明 |
|--------|------|
| `Alt+H` | 向左调整 3 格 |
| `Alt+J` | 向下调整 3 格 |
| `Alt+K` | 向上调整 3 格 |
| `Alt+L` | 向右调整 3 格 |

## 移动 Pane 到窗口

> **说明**: 使用 Shift+数字键 (特殊符号)，避免与窗口切换冲突

| 快捷键 | 目标窗口 | 实际按键 |
|--------|----------|----------|
| `Alt+!` | 窗口 1 | `Alt + Shift + 1` |
| `Alt+@` | 窗口 2 | `Alt + Shift + 2` |
| `Alt+#` | 窗口 3 | `Alt + Shift + 3` |
| `Alt+$` | 窗口 4 | `Alt + Shift + 4` |
| `Alt+%` | 窗口 5 | `Alt + Shift + 5` |
| `Alt+^` | 窗口 6 | `Alt + Shift + 6` |
| `Alt+&` | 窗口 7 | `Alt + Shift + 7` |
| `Alt+*` | 窗口 8 | `Alt + Shift + 8` |
| `Alt+(` | 窗口 9 | `Alt + Shift + 9` |

> **注意**: `Alt+1` ~ `Alt+9` 用于窗口切换，不能用于移动 pane

## 移动窗口到会话

| 快捷键 | 说明 |
|--------|------|
| `prefix` + `1` ~ `9` | 移动当前窗口到会话 1-9 |
| `prefix` + `0` | 移动当前窗口到会话 10 |

---

## 键位说明

- `prefix` = `Ctrl+s`
- `Alt` = `M-` (Meta 键)
- `Ctrl` = `C-`
- `Shift` = `S-`

## 特点

- 鼠标支持已开启
- Vim 模式键位绑定 (复制模式 + 快捷键)
- 支持真彩色 (True Color)
- 自动重编号窗口
- 历史记录限制: 10000 行
- Agent Tracker 集成 (任务追踪)
